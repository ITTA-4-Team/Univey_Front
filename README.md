## Commit Convention


| 작성 | 설명 |
| --- | --- |
| [FEAT]#이슈번호 내용 | 기능 추가 |
| [FIX]#이슈번호 내용 | 버그 및 오류 수정 |
| [DOCS]#이슈번호 내용 | 문서 수정 |

<br></br>

## Branch Convention

| 작성             | 설명              |
|----------------|-----------------|
| main           | master 브랜치      |
| develop        | main 서브 브랜치     |
| feat/#이슈번호-기능명 | 기능 추가용 feat 브랜치 |
| fix/#이슈번호-브랜치명 | 기능 수정용 fix 브랜치  |
